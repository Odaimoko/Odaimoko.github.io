（14）第十四章：这跑位不是个好东西，希望你也没有/I Hate This Run and You Should Too

假设你是个前锋，你们的边锋正在边路，刚刚进入进攻三区。她面对着对方的边后卫，正准备发起1v1单挑。你现在该干什么？

呃，你们大多数经常会这么做：就像下图所示，你斜插到角球区，把一个后卫带到那里去。现在这单挑变得有点像群架，你的持球队友得找个新计划了。这是一个将一次有威胁的进攻磨成渣渣的好方法。

让我们从小问题先开始。如果你队友把球传给了斜插到角球区的你，你接球的姿势只有一个，那就是背对整个球场（还有你所有的队友），以及一个在你脖子上呼吸的后卫。这样的话，你最好的结果是要到一个角球，更多情况下是一个界外球，还是对面的。这次跑位能得到的东西可不多。

然后是更大的问题：你完全破坏了队友1v1的机会。她跟一个后卫在一座独岛上，那个后卫无人可助，然后你过去给她帮助了。听着，1v1是一个非凡的进攻方程。如果你们的边锋是个1v1好手，那你就别给我跑过去挡她的路，让她自己完成工作。如果对手没有去掩护边后卫，不要强迫他们去。让你的队友一个人在那边，你去球门那里等着传中就好。

教练笔记

一些教练可能完全不同意我的看法，没关系，反正我是很喜欢看到我们的对手这么跑。就我所见，他们爱怎么进攻角旗怎么进攻。球员们这么跑的原因是这条路是最简单的，但那是因为这是最没有威胁的，防起来也没啥问题。如果你信我的，这样来解决你们的问题：把这张图画在黑板上，然后让你的队员们体验一下。当你的前锋还是这样跑位的时候，大声提醒她。

小田说

有那么点道理，但如果能这样跑确确实实可以创造出机会，为什么不呢？下面就是例子。 16/17赛季国王杯，皇家社会主场对巴萨，巴萨的罗贝托和拉基做了次撞墙进禁区。

这个套路本身就有个名字，叫做直传斜插。

还是那句话，判断好该跑还是不该跑。作者认为不该跑，是因为这会破坏一对一；而破坏一对一不好则是因为你的队友是个一对一好手。但在巴萨，罗贝托并不是一对一行家，他本身适合RCM的位置，所以比较倾向配合。拉基也读出了撞墙的意图，所以配合打得很漂亮。

如果队友是一对一行家，可以让他单干；如果在那里的队友是需要配合的类型（因为不一定都是边锋啦），就过去好好帮忙吧。
